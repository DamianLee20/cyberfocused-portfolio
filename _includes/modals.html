Phishing Simulation Tool

A credential harvesting simulation app built with Flask, designed to educate users on phishing threats. It captures email/password input, logs user IPs, and redirects them post-login. Ideal for security awareness training and demonstrating social engineering risks.

  • Tech Used: Python, Flask, Ngrok
  • Purpose: Simulated phishing training
  • Security Focus: Credential capture & awareness

Secure File Transfer App

Built a web application for uploading and downloading encrypted files. Files are encrypted on upload using AES encryption and can only be decrypted with a user-provided key. This project showcases file integrity and confidentiality best practices.

  • Tech Used: Flask, Python, AES (cryptography)
  • Security Focus: Encryption, secure file handling

SOC Simulation & Threat Detection

Simulated a Security Operations Center workflow by analyzing log data in Splunk and Wireshark. Detected anomalous traffic, documented incident response steps, and drafted reports using real-world log analysis practices.

  • Tech Used: Splunk, Wireshark, Tcpdump
  • Focus: Threat detection, log triage, incident response

Help Desk Simulation

This mock IT support environment involved managing Active Directory accounts, performing password resets, and documenting responses in ServiceNow. Also included Jira-based ticket tracking and process communication simulations.

  • Tech Used: Active Directory, ServiceNow, Jira
  • Focus: IT support workflows, identity and access tasks
